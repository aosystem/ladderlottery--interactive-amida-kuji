# Ladder Lottery (Amida-kuji) ⛩️🪜

A dynamic and interactive **Ladder Lottery** (known as *Amida-kuji* in Japan) built with **Flutter**. This utility provides a fair and exciting way to make random selections or assign roles for groups of 2 to 9 people.

## 🌟 Key Features
- **Dynamic Creation:** Supports group sizes from 2 up to 9 participants.
- **Shuffle Mechanic:** A dedicated shuffle button reorders start and goal positions for complete fairness.
- **Visual Progression:** Watch as horizontal rungs are drawn in real-time and markers travel down the ladder to reveal the results.
- **Global Support:** Fully localized in **50+ languages**, making this traditional Asian selection method accessible to everyone.

## 💡 Best For...
- **Decision Making:** Deciding who buys lunch, takes out the trash, or leads a project.
- **Game Nights:** Assigning roles or teams in a fun, visual way.
- **Event Raffles:** Small-scale drawings with a "suspenseful" reveal.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Implementation of the ladder-crossing algorithm and real-time path animation using Flutter's drawing capabilities.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
